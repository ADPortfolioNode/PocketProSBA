# Lender Match
Source: SBA API endpoint response
API route: /api/sba/content/loans/lender-match

Before you start talking to lenders, have a look at the abbreviated checklist below to see if you're ready for a traditional SBA loan. We will prepare a summary of all the interested lenders two business days after you submit a Lender Match request. From there, you can start talking to lenders and/or complete the application they provide. Some will reach out to you, and you’re welcome to contact them as well.
